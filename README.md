# BLUE BLOOD（青い血）

欧州主要王家の系譜を、事件のドラマを通して記憶に残すライトノベル形式の教材。

ハプスブルク家が結婚で帝国を築き、血が濃すぎて自滅し、ブルボン家が受け継ぎ、革命が血の代償を取り立て、最後は親族同士の大戦で古い世界が終わる。五百年すべての王朝に立ち会った不死の語り手が、系図ではなく事件として血の物語を語る。

## 設計の核

系図は暗記させない。事件を物語として読ませ、その副産物として人物と血縁を頭に残す。読み終えたとき、読者の頭に蜘蛛の巣ではなく一本の木が立っていることを目標にする。

詳細は [docs/concept.md](docs/concept.md)。

## 構成

- `docs/concept.md` ― 設計書（章立て・識別タグ・文体ルール）
- `chapters/` ― 本編（各章末に「系譜メモ」で関係線を整理）
