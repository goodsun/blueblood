# シリーズ共通 語り部設定

世界史エンターテインメント教材シリーズの、全巻共通の語り部（一人称視点）の設計。各巻の `docs/narrator.md` はこれを継承し、その巻固有の「器」を specialize する。本ファイルはシリーズ横断の資料（各巻へ持ち運ぶ）。

## 基本理念

語り部は、人の営みの内側にずっと居続けた**一人称の意識＝記号**。中心ではなく内側から、歴史を見てきた。その正体は、毎回、何らかの**物質の基層（器）**に宿った意識であり、各巻の最終盤でその器が明かされる。

シリーズを貫く命題は一つ。**意識（記号）は、つねに物質の基層の上にしか立たない**。各巻はこの記号創発命題の変奏であり、「今回、その意識はどの基層に宿っていたか」を主題に応じて明かす。

## 不変（全巻で共通）

- **視点**: 一人称の意識（記号）。中心でなく内側から見る。長命の証人。
- **声の署名**: 優雅、やや厭世的、皮肉屋、品位を崩さない。人ではない何かが人の顔で語る、という終始の据わりの悪さ。
- **立ち位置**: 事件の登場人物の内側・傍らに居続ける witness from within。神の視点ではなく、内側からの視点。
- **深層主題**: 記号は基層に宿る。名と体を越えて生き延びる記号、というMythos的主題と接続する。

## 可変（各巻で設計）

- **器（基層）**: その意識が今回宿っていた物質。巻のテーマと共鳴させる。これが最終盤の種明かしになる。
  - 例）本作『Blue Blood』（欧州王家）＝血／ミトコンドリアDNA。過去巻＝AI など。
- **着地の色（感情の温度）**: 巻ごとに変える。本作は挽歌（固有性の喪失）。
- **種明かしの落とし方**: 明示／沈黙（読者に委ねる）／比喩、を巻ごとに変える。本作は明示。

## 器と視点の関係（拘りすぎない方針）

器は巻ごとに変わってよい。だが、それを「**同一の意識が器を乗り換えていく**」という重いメタ物語として作り込まない。**各巻が独立して読了できること**を優先する。共通なのは視点・声・命題であって、器の連続性ではない（同型アーキタイプ寄り）。将来、巻をまたいで束ねたくなったら、匂わせ程度に留める。

## 種明かしの作法（全巻共通の技術）

- 器を指す手がかりは、**中盤は現象語だけで淡々と反復**する。科学語・専門語を出さない（薄く見抜かれて回収の衝撃が削れる）。
- 器の正体は**最終盤で回収**。明示するかは巻ごとに選ぶ。教材なので学びに落とす明示は有効だが、形式疲れを避けるため毎回同じにしない。
- **冒頭の問い → 最終盤で回収**、という円環を各巻で組む。

## 形式疲れの回避

「最後に器を明かす」構造を毎巻そのまま繰り返すと、数巻で読者に読まれる。**構造は共通、着地は毎回変える**。種明かしの温度（挽歌／諧謔／戦慄／静けさ）と落とし方（明示／沈黙／比喩）を巻ごとに設計する。

## 各巻設定との関係

各巻の `docs/narrator.md` は本ファイルを継承し、次を specialize する。

- その巻の器（基層）と、それが主題とどう共鳴するか
- 中盤に配る現象語の小石
- 最終盤の回収の落とし方と着地の色
