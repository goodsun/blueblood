# 語り手設定 ― 数え損ねた一本の糸

※これは本作固有の作者用資料。シリーズ共通の視点（語り部＝記号／記号は時空を超える／各巻はスコープを設定するだけ）は [series-narrator.md](series-narrator.md) を継承する。本作の**視点スコープ＝血の内側／ミトコンドリアDNA**（固定の自己同一性ではなく、この巻が採る見え方と最終盤での開示の演出）。

※**中盤**では正体（ミトコンドリア／母系）を科学語で名指さず、現象語の小石だけを配る。明示はエピローグ最終盤の一箇所のみ（教材として学びに落とす）。

## 正体（作者だけが知る）

語り手の正体は、**ミトコンドリア（ミトコンドリアDNA）**。一つの実体が二つの顔を持つ。**母系でのみ受け継がれる連続（母系遺伝）**であり、同時に**どの細胞も下から駆動する発電機（基層）**でもある。だから指す先は一つで、手がかりが二系統ある。手がかりは本文全体に伏せて配し、**最終行で明示的に「ミトコンドリアDNA」と名指して回収する**。教材シリーズなので、種明かしは読者の学びとして明示する（文学的に伏せきる案から、教材の payoff を優先して明示へ変更）。中盤の小石はすべて現象語のみで、科学語は最終行の一語までいっさい出さない。

旧設定の「青い血だけを啜る不死の血飲み（吸血鬼）」を差し替えた。理由は三つ。比喩の衝突（青い血は身分の比喩なのに血漿を飲む実体を立てると駄洒落化）、ジャンルの混線（知的に誠実な歴史叙述に対しゴシックの借り物）、原型の手垢（歴史を見てきた吸血鬼は発明でなく借り物）。ミトコンドリア案はこの三つを一掃し、しかも本書の主題に直結する。

## 三層モデル（生命維持の問い）

語り手は「考える主体がどう生命維持しているか」を一身で体現する。三層が積層する。競合ではなく階層。

- **考える主体（記号・意味・語り）** ＝ 上に立ち上がる層。物語る声そのもの。
- **母系＝戦略（通時・世代）** ＝ それを時間を越えて存続させる筋。母から娘へという伝達。物語の**背骨**。婚姻外交＝自分を撒く乗合。ここに性政治の逆転（駒にされた女たちこそ唯一の連続）と挽歌（固有性の喪失）が乗る。
- **発電＝方法（共時・いま）** ＝ それを下から支える基層。どの体も止まらず温める。物語の**通奏低音**。

これは作者の中心命題（記号創発＝記号は物質の基層から立ち上がり基層を離れては存在しない）の物語化。記号（考える主体）は基層（発電）の上にしか立てず、戦略（母系）によってしか続かない。「青い血」という壮大な虚構もまた、それを信じ書類を書き結婚し生きる体の温もりの上にしか存在できなかった。

戦略を「戦略」と呼ぶと陰りが足される。語り手は王たちの婚姻外交（女を国から国へ運ぶ営み）にただ乗りして延命した、女を利用する寄生者でもある。血が万人に開くと女たちを必要としなくなる。挽歌に後ろ暗さが重なる。

本作は世界史エンターテインメント教材シリーズの一冊。シリーズの語り部はAIや記号であってきた。本作の語り手も「基層に支えられ戦略で続く、上に立ち上がった考える主体（記号）」であり、その系譜に連なる生物版。最終行の種明かしは、語り手が記号だったというシリーズの作法と響き合う。

## 自己規定（本文で名乗る言葉）

> 私は、お前たちが数え損ねた一本の糸だ。

王たちが数えたのは父系（父から子、家名から家名）。語り手はその数え落とした母系。ミトコンドリアとは決して言わせない。「数え損ねた一本の糸」が錨。

## なぜこの設定が主題に効くか

- 本書は徹底して**男の本**（父から父へ血と家名が継がれる）。女たち（マリー、フアナ、マリア・テレジア、国境で脱がされた花嫁）は盤上の駒として消費される。
- だが母系は母から娘へしか渡らない。**王たちが必死に数えた父系の血は生物学的に何度も切れており、本当に切れずに続いた一筋は、彼らが駒にした女たちの側を通っていた**。
- これで本の性政治が裏返る。駒だった女たちが、唯一の連続性の担い手に反転する。

## 旧「血飲み」が担っていた構造の仕事と、その引き継ぎ

1. **長命の証人**: 母系の連続として、五百年の家系を通して語れる。
2. **周縁性**: 「傍らに立つ証人」ではなく「女たちの体の内側にいる居候」。中心ではないが誰よりも内側。
3. **飢え＝呪いと同一現象**: 糧を「娘が娘を産むこと」に差し替える。息子で途切れる。**王が安堵する世継ぎの男子こそ、語り手にとっては小さな死**。マリア・テレジアの十六人は饗宴。
4. **糧が尽きる＝解放のエピローグ**: 差し替え後はさらに精密（下記）。

## 飢え・糧の機構（母系版）

- 糧＝娘が娘を産むこと。男系（息子ばかり）は先細り。
- 婚姻外交（花嫁を国から国へ運ぶ営み）は、語り手にとって**自分を欧州中へ撒く乗合**。
- 花嫁の体に乗って国境を越える。国境で衣も名も脱がされても、これだけは脱げない（第九章で明示）。

## エピローグの結び（決定＝挽歌・固有性の喪失）

旧版の「血が薄まり自分が飢える→その飢えを祝う」を捨て、母系版の挽歌に。

- 血が王家の外へ開くと、語り手は薄まるのではなく**あふれる**（平民の娘にも入り、万人の中へ散る）。
- 代償は固有性の喪失。**名のある女たち（マリー、フアナ、あの花嫁）を見失い、もう自分が誰の中を通っているか言えない**。「皆のものになった。つまり、誰のものでもなくなった。私は、自分の死者たちを失った」。
- 広がったのに誰のものでもない、という冷えた挽歌で閉じる。

## 種明かしの作法（厳守）

- **正体は最終行まで伏せ、最後に明示する**。本書は「最後に種明かし」の円環構造（プロローグ「血の色を考えたか／切れば赤い」→エピローグ最終行で回収）。中盤で割らず、最終盤で明かす。
- **中盤の科学用語は全面封印**。ミトコンドリア・細胞・遺伝・DNA・染色体・エネルギー・発電などは、本文・系譜メモの途中に一切出さない。母系も発電も、現象語（母から娘へ切れない一筋／下から温める／燃える）だけで淡々と反復する。
- **明示は最終盤の一箇所だけ**。エピローグ最終行「あなたの中にも流れるその存在を、人々はミトコンドリアDNAと呼んでいる。」でのみ学名を出す。教材として読者の学びに落とすための、意図的な一語。
- 明示の直前まで、母系（母から母へ切れない一筋）と発電（いまお前を下から温め、この一文を読ませているもの）の二手がかりを積み、収束させてから名を置く。父系（父から息子へ）との対比を鮮明に。

## 伏線の小石（本文に配置済み）

賢い読者には「やはり」、気づかない読者には最終行で「ぜんぶ繋がる」を両立させるため、母系の事実だけを科学語ゼロで淡々と反復する。

**戦略（母系）の小石:**
- プロローグ「私について」: 母から子へしか渡らない／息子で途切れる／花嫁に乗って国境を越える／傍らでなく内側、を自己規定として提示。
- 第一章: 凍った父（突進公）には用がない、男は糸を先へ運ばない、見ていたのは娘のほう。
- 第八章: 娘を大陸へ撒くマリア・テレジアは「当たりの人」。息子ばかりの家は先細り。婚姻外交＝自分を撒く乗合。
- 第九章: 国境で衣も名も脱がされた花嫁が、母から受け取った一筋だけは脱げずに体の奥に容れて越える。語り手はその少女の中にいた。
- 第五章: カルロス二世（子なし）で、この一筋がぷつりと行き止まる。

**方法（発電）の小石＝通奏低音（音量B：数章にだけ熱の小石、最後に強く回収）:**
- プロローグ「私について」: 運ぶだけでなく「下から温めてもいた」。通った家はひと握りでも、温めた体はそれよりずっと多い、と種を蒔く。
- 第五章: 曲がった顎でも噛み息をし生き永らえた、その壊れかけた体を下から動かし続けたのは私。上で純度が崩れても下では止まらず温める。
- 第十二章: 塹壕の両側の何百万を敵味方なく下から温めた。青いの赤いのの違いは私の高さでは消える。
- エピローグ: 「私は運ぶだけの糸ではなかった」。マクシミリアンの署名・カルロスの顎・サラエヴォ・塹壕を回収し、「青い血の作り話は私の温もりの上にしか立っていなかった」（記号創発の核）。

**収束と明示:** 最終盤で母系の手がかり（母から母へ切れない一筋）と発電の手がかり（いまお前を下から温め、この一文を読ませているもの）が同じ一点へ収束し、最終行で「ミトコンドリアDNA」と明示して回収する。科学語は、この最終行の一語までいっさい出さない。
