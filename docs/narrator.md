# 語り手設定 ― 数え損ねた一本の糸

※これは作者用の内部資料。**本文・系譜メモでは、ここに書く正体（ミトコンドリア／母系遺伝）を絶対に名指さない**。読者に明かすのは最終行の「定義」だけ。

## 正体（作者だけが知る）

語り手の正体は、**母系でのみ受け継がれる連続＝ミトコンドリア（母系遺伝）**。母から子へは渡るが、息子からは次へ渡らない。父系の家系図では決して数えられない、もう一本の血の線。

旧設定の「青い血だけを啜る不死の血飲み（吸血鬼）」を差し替えた。理由は三つ。比喩の衝突（青い血は身分の比喩なのに血漿を飲む実体を立てると駄洒落化）、ジャンルの混線（知的に誠実な歴史叙述に対しゴシックの借り物）、原型の手垢（歴史を見てきた吸血鬼は発明でなく借り物）。母系案はこの三つを一掃し、しかも本書の主題に直結する。

## 自己規定（本文で名乗る言葉）

> 私は、お前たちが数え損ねた一本の糸だ。

王たちが数えたのは父系（父から子、家名から家名）。語り手はその数え落とした母系。ミトコンドリアとは決して言わせない。「数え損ねた一本の糸」が錨。

## なぜこの設定が主題に効くか

- 本書は徹底して**男の本**（父から父へ血と家名が継がれる）。女たち（マリー、フアナ、マリア・テレジア、国境で脱がされた花嫁）は盤上の駒として消費される。
- だが母系は母から娘へしか渡らない。**王たちが必死に数えた父系の血は生物学的に何度も切れており、本当に切れずに続いた一筋は、彼らが駒にした女たちの側を通っていた**。
- これで本の性政治が裏返る。駒だった女たちが、唯一の連続性の担い手に反転する。

## 旧「血飲み」が担っていた構造の仕事と、その引き継ぎ

1. **長命の証人**: 母系の連続として、五百年の家系を通して語れる。
2. **周縁性**: 「傍らに立つ証人」ではなく「女たちの体の内側にいる居候」。中心ではないが誰よりも内側。
3. **飢え＝呪いと同一現象**: 糧を「娘が娘を産むこと」に差し替える。息子で途切れる。**王が安堵する世継ぎの男子こそ、語り手にとっては小さな死**。マリア・テレジアの十六人は饗宴。
4. **糧が尽きる＝解放のエピローグ**: 差し替え後はさらに精密（下記）。

## 飢え・糧の機構（母系版）

- 糧＝娘が娘を産むこと。男系（息子ばかり）は先細り。
- 婚姻外交（花嫁を国から国へ運ぶ営み）は、語り手にとって**自分を欧州中へ撒く乗合**。
- 花嫁の体に乗って国境を越える。国境で衣も名も脱がされても、これだけは脱げない（第九章で明示）。

## エピローグの結び（決定＝挽歌・固有性の喪失）

旧版の「血が薄まり自分が飢える→その飢えを祝う」を捨て、母系版の挽歌に。

- 血が王家の外へ開くと、語り手は薄まるのではなく**あふれる**（平民の娘にも入り、万人の中へ散る）。
- 代償は固有性の喪失。**名のある女たち（マリー、フアナ、あの花嫁）を見失い、もう自分が誰の中を通っているか言えない**。「皆のものになった。つまり、誰のものでもなくなった。私は、自分の死者たちを失った」。
- 広がったのに誰のものでもない、という冷えた挽歌で閉じる。

## 種明かしの作法（厳守）

- **正体は最終行まで伏せる**。本書はそもそも「最後に種明かし」の円環構造（プロローグ「血の色を考えたか／切れば赤い」→エピローグ最終行で回収）。語り手の正体だけ途中で割らない。
- **科学用語を本文・系譜メモで全面封印**。ミトコンドリア・細胞・遺伝・DNA・染色体・エネルギー・十数億年などは一切出さない。近接語も封印（薄く見抜かれて点灯の衝撃が削れる）。
- **最終行は定義だけで読者に渡す**。「父から父への血ではない。お前の母の、そのまた母の中を、一度も途切れずに流れてきた一筋。お前もいま母から受け取っている。名前は、自分で調べるといい」。
- 学名を語り手に名乗らせない（時代の声と理科の register の衝突を避ける）。

## 伏線の小石（本文に配置済み）

賢い読者には「やはり」、気づかない読者には最終行で「ぜんぶ繋がる」を両立させるため、母系の事実だけを科学語ゼロで淡々と反復する。

- プロローグ「私について」: 母から子へしか渡らない／息子で途切れる／花嫁に乗って国境を越える／傍らでなく内側、を自己規定として提示。
- 第一章: 凍った父（突進公）には用がない、男は糸を先へ運ばない、見ていたのは娘のほう。
- 第八章: 娘を大陸へ撒くマリア・テレジアは「当たりの人」。息子ばかりの家は先細り。婚姻外交＝自分を撒く乗合。
- 第九章: 国境で衣も名も脱がされた花嫁が、母から受け取った一筋だけは脱げずに体の奥に容れて越える。語り手はその少女の中にいた。
- 第五章: カルロス二世（子なし）で、この一筋がぷつりと行き止まる。
- エピローグ: 定義のみで正体を示し読者に渡す。

## 声

優雅、やや厭世的、皮肉屋。品位を崩さない。文体ルール（ダッシュ不使用・読点過多回避・年号算用数字・代数漢数字）に準拠。正体を伏せるぶん、「人ではない何かが人の顔で語っている」据わりの悪さを終始まとわせ、最終行で読者が遡って総毛立つことに賭ける。

## レトロフィット記録（旧血飲み→母系）

- プロローグ「私について」全面改稿（飲む→中にいた、傍ら→内側、自己規定を糸に）。
- 第一章「血を舐めた」描写を削除し母系の手がかりに。
- 第五章 死の床の「啜る／糧」を「この一筋が行き止まる」に。
- 第八章・第九章に小石を追加。
- 第十二章「血の傍ら」→「血の内側」。エピローグ参照の自己言及を母系に。
- エピローグ「私について、もう一度」「最後の着地」「系譜メモ」を母系・挽歌・定義回収に全面改稿。
