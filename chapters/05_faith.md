# 第五章　信仰という二本目の線

## もう一本の線

　ここまで、私は一本の線だけを追ってきた。血の線だ。誰が誰と結婚し、誰が何を相続したか。だが正直に言おう。この物語には、もう一本、別の線が走っている。その線が、血の線をあちこちで断ち切り、ねじ曲げてきた。

　信仰の線だ。

　ここで一度、その線を見せておかねばならない。これを知らないと、この先の章で起きることの本当の理由がわからないからだ。なぜスペイン・ハプスブルクが、あれほど身内とばかり結婚したのか。その答えの半分は、血ではなく、信仰のほうにある。

## 一人の修道士

　1517年。ドイツの片隅で、一人の修道士が教会の扉に紙を貼った。マルティン・ルター。貼ったのは、当時の教会への抗議文だった。

　はじめは小さな出来事に見えた。一人の修道士の不満。だが活版印刷が、その紙をまたたく間にヨーロッパ中へ運んだ。第一章でマクシミリアンが自分の宣伝に使った、あの新しい機械だ。同じ機械が、今度は教会を割る道具になった。

　数年のうちに、西方のキリスト教は二つに裂けた。古いカトリックと、新しいプロテスタント。ローマ教皇に従う者と、従わない者。それまで一つだった信仰が、二つになった。

　これが、王家にとって何を意味したか。並の出来事ではない。これは、結婚というものの土台を、根こそぎ作り替えた。

## 結婚市場が二つに割れる

　ルター以前、カトリックの王子は、どこのキリスト教の王女とでも結婚できた。相手はみな同じ信仰だったからだ。ヨーロッパ中の青い血が、一つの大きな池に入っていた。誰もが、その池から相手を選べた。

　ルター以後、その池が二つに割れた。

　カトリックの池と、プロテスタントの池。そして、この二つは混じらなくなった。カトリックの王家は、プロテスタントの王女を娶らない。プロテスタントの王家も、カトリックの相手を選ばない。信仰が違えば、もう同じ食卓につけない。同じ寝室にも入れない。

　考えてみてほしい。それまで大陸全体だった結婚相手の池が、半分になったのだ。選べる相手が、半分に減った。とりわけ、自分の信仰を国の柱と決めた家ほど、相手探しは窮屈になった。

　この「池が割れた」という一事が、これから先のすべての結婚を縛る。覚えておいてほしい。血の線は、この日から、信仰の線の内側でしか動けなくなった。

## カール五世の見果てぬ夢

　この裂け目の真上に立たされたのが、第二章の男だ。全部相続した男、カール五世。

　彼はカトリックの守護者だった。皇帝として、割れた信仰をもう一度一つに戻すのが務めだと信じていた。だから彼は、ルターと正面から向き合った。1521年、帝国議会にルターを呼び出し、説を撤回させようとした。だがルターは退かなかった。

　以後、カールの生涯は、この裂け目を縫い合わせる戦いに費やされる。だが縫っても縫っても、裂け目は広がった。ドイツの諸侯のなかに、新しい信仰を選ぶ者が次々に現れたからだ。彼らにとって、プロテスタントを選ぶことは、皇帝の手から自分の領地を引き離す口実でもあった。信仰と領地は、いつも裏で手を結ぶ。

　1555年、カールはついに諦めた。アウクスブルクの和議だ。決められたのは、こうだった。それぞれの領主が、自分の領地の信仰を決めてよい。領主がカトリックならその地はカトリック、プロテスタントならプロテスタント。

　一見、賢い妥協に見える。だが中身は、降伏だった。これは、ドイツが二つの信仰に永久に割れることを、皇帝が認めた文書だ。一つに戻す夢は、ここで死んだ。

　第三章で、カールが疲れ果てて退位した話をした。その疲れの半分は、この見果てぬ夢の重さだった。彼は世界一の帝国を持ちながら、自分の足元のドイツの信仰一つ、元に戻せなかった。和議の翌年、彼は冠を下ろす。

## 窓から落ちた使者たち

　信仰の裂け目は、和議では塞がらなかった。半世紀後、それは大陸最大の戦争になって噴き出す。

　1618年、ボヘミアのプラハで、事件が起きた。新教徒たちが、カトリックの王の使者を、城の窓から放り投げたのだ。プラハ窓外投擲と呼ばれる。窓から人が落ちた、ただそれだけの出来事が、合図になった。

　ここから三十年、ヨーロッパは戦い続ける。三十年戦争だ。

　はじめはカトリックとプロテスタントの宗教戦争だった。だが戦ううちに、信仰の名目は薄れ、各国の領地と利害の奪い合いに変わっていった。戦場の多くは、ハプスブルクの膝元、ドイツと中欧だった。傭兵の群れが村々を焼き、奪い、病をまき散らした。ドイツの人口は、地域によっては三分の一が消えたとも言われる。私が見てきた戦争のなかでも、これは底なしに惨かった。

　1648年、ようやく戦いが終わる。ウェストファリアの講和だ。この条約は、信仰の地図を確定させ、同時に、国というものの新しい形を決めた。それぞれの国が、上に立つ皇帝にも教皇にも縛られず、自分の領土を自分で治める。今日まで続く主権国家の仕組みが、ここで形になった。

　そしてハプスブルクは、力ずくで信仰を一つに戻す試みに、完全に失敗した。割れた信仰は、割れたまま固定された。

## 閉じていく池

　さて、ここまでの話を、血の線に結び直そう。これが、この章でいちばん大事なところだ。

　信仰の裂け目は、ハプスブルクとスペインを、対抗宗教改革の先頭に立たせた。彼らはカトリックの旗手になった。骨の髄までカトリック。それは誇りだった。だが、その誇りには、見えない代償がついていた。

　彼らは、プロテスタントの相手と結婚できない。だから、選べる花嫁は、カトリックの王家に限られる。そして、格の釣り合うカトリックの大王家など、ヨーロッパにいくつもない。数えるほどだ。フランスのブルボンは長く宿敵で、すんなりとは結べない。残るのは、結局、自分の身内だった。もう一方のハプスブルク。同じ家の、いとこ。姪。

　信仰が、結婚相手の池を狭めた。狭まった池の底に残っていたのは、自分自身の血だった。だからスペイン・ハプスブルクは、いとこと、姪と、繰り返し結婚した。誇りと信仰が、二人がかりで、血を内へ内へと押し込めた。

　血を煮詰めたのは、ケチや偶然ではない。信仰だ。混じり物のないカトリックであろうとすればするほど、混じり物のない血のなかへ閉じこもるしかなかった。プロローグの「青を絶やすまいとした」に、ここで「信仰を曲げまいとした」が重なる。二本の線が、一つの場所で絡み合う。

　その絡まりの果てに、何が生まれたか。一人の王の、ねじれた体だ。だが、その請求書を見にいく前に、同じ裂け目を海の向こうから眺めておきたい。信仰の線は、大陸のハプスブルクを身内の池へ閉じこめる一方で、一つの島を、まるごとカトリックの池の外へ押し出していた。

　次の章は、その島の話だ。海を渡った、一つの離婚から始めよう。

## 系譜メモ

この章で覚えるのは、人物ではなく、本書の二本目の縦糸「信仰」と、それが血に与えた力。

### 押さえる出来事
- **宗教改革（1517年〜）**: ルターの抗議から西方キリスト教がカトリックとプロテスタントに分裂。活版印刷が拡散を加速（第一章の機械の再登場）。
- **アウクスブルクの和議（1555年）**: 「領主が領地の信仰を決める」。ドイツの宗教分裂を皇帝が公認した降伏文書。カール五世退位（第三章）の疲れの半分はこの失敗。
- **三十年戦争（1618〜1648年）**: プラハ窓外投擲に始まる大陸最大の宗教戦争。戦場は主にドイツ・中欧、人口が激減。
- **ウェストファリアの講和（1648年）**: 信仰の地図を確定し、主権国家の仕組みを生む。ハプスブルクの「信仰を一つに戻す」試みは完全に失敗。

### つながる一本の理屈（最重要）
```
宗教改革 → 結婚市場が二つに割れる（カトリックの池／プロテスタントの池・混じらない）
        → カトリックの旗手ハプスブルクは相手の池が狭まる
        → 格の釣り合うカトリック大王家はわずか／仏ブルボンは宿敵
        → 残る相手は身内（いとこ・姪）
        → いとこ婚・おじ姪婚の反復
        → 第七章「呪われた顎」
```

### 覚えどころ
- 本書には**血の線**と**信仰の線**の二本がある。信仰の線は、血の線をあちこちで断ち切り、ねじ曲げる。
- 最重要の因果：**宗教改革が結婚市場を二分し、対抗宗教改革の旗手ハプスブルクのカトリックの結婚池を狭め、いとこ婚を強制した**。近親婚の原因の半分は信仰。プロローグ「青を絶やすまいとした」に「信仰を曲げまいとした」が重なる。
- 三十年戦争は、ハプスブルクが力で信仰を一つに戻そうとして失敗した戦争。割れた信仰は固定された。
- 第七章「呪われた顎」は、この閉じた池の最終的な請求書。
