# 第八章　十六人産んだ女帝

## 娘では継げない

　話をウィーンへ戻そう。第六章で、スペインを諦めて皇帝になった男がいた。カール六世だ。彼には、一つだけ片付かない悩みがあった。

　息子がいなかった。

　ハプスブルクの広大な領地は、これまで男が継いできた。だがカール六世には、生き残った子が娘しかいなかった。長女のマリア・テレジアだ。このままでは、自分が死んだとき、女に継がせていいのかという問題が残る。

　そこでカール六世は、何年もかけて、ある文書を各国に認めさせて回った。プラグマティッシェ・ザンクツィオン。国事詔書と訳される。中身を一言で言えば、こうだ。私の娘マリア・テレジアが、ハプスブルクの全領地を継ぐ。それを皆で認めよ、という約束だった。

　彼は生涯をかけて、この約束をヨーロッパ中から取りつけた。これで娘の相続は安泰だ、と信じて。だが彼は、第二章の教訓を忘れていた。紙の約束は、棺の前では軽い。彼が死んだ瞬間に、各国はどう動いたか。

## 約束は破られる

　1740年、カール六世が死んだ。マリア・テレジアが、ハプスブルクの全領地を継いだ。二十三歳の若い女性だった。

　その途端、約束は破られた。

　あれほど各国に認めさせた国事詔書を、皆が次々に反故にした。なかでも素早かったのが、プロイセンの若き王フリードリヒだ。後に大王と呼ばれる男。彼は約束など歯牙にもかけず、ハプスブルクの豊かな領地シュレジエンに軍を入れ、奪い取った。

　第一章を思い出してほしい。父を亡くし、世界一の遺産を抱えて孤立したブルゴーニュのマリー。あのとき、若い相続人の財産に、周りの男たちが群がった。今また同じことが起きた。父を亡くし、広大な遺産を抱えた若い女に、周りの国々が群がった。二百六十年経っても、構図は変わらない。豊かな遺産を継いだ若い相続人は、それだけで狙われる。

　だがマリア・テレジアは、マリーとは違った。彼女は屈しなかった。

　彼女はハンガリーへ赴き、幼子を抱いて貴族たちに助けを求めた。誇り高い男たちの心を動かし、兵を引き出した。奪われた領地のすべては取り返せなかったが、ハプスブルクの相続そのものは守りきった。群がる男たちの前で、若い女が遺産を守り抜いた。私はその気概を、長い記憶のなかでも上等な部類として覚えている。

## 名前が変わる

　ここで、一つ大事な変化が起きる。家の名前が変わるのだ。

　マリア・テレジアは女性だった。だから、彼女が産む子は、夫の家名を名乗ることになる。彼女の夫は、ロートリンゲン家のフランツだった。ロレーヌの公だ。二人は珍しく、政略よりも情で結ばれた夫婦だった。

　その結果、ハプスブルクの血を引きながら、名乗りはロートリンゲンを加えた家になる。ハプスブルク=ロートリンゲン家だ。

　名前を聞くと別の家のようだが、中身は続いている。マリア・テレジアの血、すなわちハプスブルクの血が、夫の家名をまとって続いていく。この先に出てくるオーストリアの皇帝たちは、みなこのハプスブルク=ロートリンゲン家だ。純血が枯れたスペイン・ハプスブルクとは対照的に、ウィーンのほうは新しい血と名を入れて、生き延びた。血を開いた家は残り、血を閉じた家は絶える。第五章と並べると、その対比がよく見える。

## 子という外交

　マリア・テレジアは、十六人の子を産んだ。

　女帝でありながら、これだけの数の子を産み育てた。そして彼女は、その子供たちを、ただ育てたのではない。第一章のハプスブルクの戦法を、彼女は industrial な規模で甦らせた。子を、ヨーロッパ中へ嫁がせ、婿入りさせる駒として使ったのだ。

　戦争は他家に任せよ、結婚せよ。あの一族の合言葉を、彼女は十六人ぶんの縁談で実践した。娘をナポリのブルボン家へ。別の娘をパルマへ。そして、とりわけ大きな一手があった。長年の宿敵だったフランスのブルボン家と、手を結ぶことにしたのだ。

　ハプスブルクとブルボンは、第六章まで、スペインを奪い合った敵同士だった。その二大家が、プロイセンという新しい脅威を前に、和解しようとした。外交革命と呼ばれる、大きな方針転換だ。そして和解を血で固めるために、マリア・テレジアは末の娘の一人を、フランスの王太子のもとへ送り出す。

　二百年以上、戦い続けたハプスブルクとブルボン。その二つの青い血が、ついに一組の夫婦のなかで一つになる。これは、本書がずっと描いてきた二つの主役の家が、最後に交わる瞬間だ。

　送り出される娘の名は、マリア・アントニア。フランスへ渡れば、その名はフランス風に変わる。マリー・アントワネット。

　母は、娘を平和の使者として送り出した。その娘が、二十数年後に何を迎えるかを、まだ誰も知らない。

　その話は、次の章でしよう。

## 系譜メモ

この章でつながる関係線は一本。**男子のないハプスブルクをマリア・テレジアが継ぎ、ハプスブルク=ロートリンゲン家として再生し、子を使った婚姻外交でブルボンと結ぶまで**。

### 登場人物と一行タグ
- **カール六世**: スペインを諦めた皇帝。男子がなく、娘の相続を認めさせる「国事詔書（プラグマティッシェ・ザンクツィオン）」を各国から取りつける。
- **マリア・テレジア**: 十六人産んだ女帝。父の死で二十三歳で全領地を継ぎ、約束を破った各国の攻撃（オーストリア継承戦争）を耐えて相続を守る。
- **フリードリヒ（プロイセン王・大王）**: 約束を無視してシュレジエンを奪取。ハプスブルクの新しい強敵。
- **フランツ（ロートリンゲン家）**: マリア・テレジアの夫。情で結ばれた相手。家名が「ハプスブルク=ロートリンゲン」になる由来。
- **マリー・アントワネット**: マリア・テレジアの末娘の一人。仏ブルボンとの和解（外交革命）を固めるためフランス王太子へ嫁ぐ（次章）。

### つながった一本の線
```
カール六世（男子なし・国事詔書）
      │
マリア・テレジア × フランツ（ロートリンゲン家）
      │   → 家名が「ハプスブルク=ロートリンゲン」に
      │（十六人の子を婚姻外交の駒に）
      ├ ヨーゼフ二世・レオポルト二世（後の皇帝）
      ├ 娘 → ナポリのブルボン家ほか各地へ
      └ マリー・アントワネット → フランス王太子（後のルイ十六世）へ
                                  ＝ハプスブルクとブルボンが一つに
```

### 覚えどころ
- マリア・テレジア＝**唯一の女性ハプスブルク君主にして十六人の母**。父の死で約束を破られ攻め込まれるが、相続を守り抜く（第一章のマリーと同じ「狙われる若い相続人」構図、ただし守りきった点が逆）。
- 女性のため家名が変わる：**ハプスブルク → ハプスブルク=ロートリンゲン**。新しい血と名を入れて存続。**血を閉じたスペインは絶え、血を開いたウィーンは残る**（第五章との対比）。
- 子十六人を婚姻の駒に使い、宿敵ブルボンと和解（**外交革命**）。その総仕上げが末娘マリー・アントワネットのフランス輿入れ。
- ここで**本書の二大家ハプスブルクとブルボンがついに一つの結婚で交わる**。次章「オーストリアから来た花嫁」で、その帰結（断頭台）へ。
